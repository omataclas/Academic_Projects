Django é Popular

O web-framwork Django é usado por sites muito populares, como:

  • Instagram
  • Youtube
  • Spotify
  • DropBox
  • Pinterest

Django tornou se um projeto de código aberto após ser criado como um sistema para gerenciar um site jornalístico na cidade de Lawrence, no Kansas. Hoje, o Django é um framework para desenvolvimento rápido para web, escrito em Python e utilizando o padrão model-template-view.
